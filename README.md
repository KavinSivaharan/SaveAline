# SaveAline

A knowledge base scraper and content management system built with React, TypeScript, and Supabase.

## Features

- Web scraping functionality to pull content into knowledge base format
- Modern React application with TypeScript
- Supabase backend integration
- Beautiful UI built with shadcn-ui and Tailwind CSS

## Setup

Follow these steps to get started:

```sh
# Step 1: Clone the repository
git clone https://github.com/KavinSivaharan/SaveAline.git

# Step 2: Navigate to the project directory
cd SaveAline

# Step 3: Install the necessary dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Tech Stack

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Development

```sh
npm run dev      # Start development server
npm run build    # Build for production
npm run lint     # Run linter
npm run preview  # Preview production build
```
